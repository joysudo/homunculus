This is a landing page/proof of concept for a coding program named **Homunculus**. Participants learn neuroevolution and get prizes.

The site is built using Sveltekit and deployed on web at [homunculus-two.vercel.app](https://homunculus-two.vercel.app/). Illustrations drawn on Procreate, inspired by linocut prints. One of my first implementations of CSS parallax and scripted confetti with custom SVG paths!
